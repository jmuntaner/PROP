DriverPuntuacioProblema

Executable i jocs de proves per a provar la classe PuntuacioProblema

Fitxers:
    - DriverPuntuacioProblema.jar:  Executable del driver de PuntuacioProblema.
    - jp_in:            Entrada del joc de proves.
    - jp_out:           Sortida esperada del joc de proves.
    - jp_explicat.txt:  Descripcio dels jocs de proves.
    - README.md         Markdown amb l'operativa.

Operativa:
  - Execucio interactiva:

      java -jar DriverPuntuacioProblema.jar

    Es presenta un menu amb opcions per a utilitzar les diferents parts de la classe.

  - Execucio jocs de proves:

      java -jar DriverPuntuacioProblema.jar -q < jp_in.txt > sortida.txt

    Per a comprovar la validesa del resultat, executar:

      diff sortida.txt jp_out.txt

    Si no produeix sortida, el test s'ha superat amb exit.

DriverJugar

Driver de les funcionalitats principals del primer lliurament

Fitxers:
    - DriverJugar.jar:  Executable amb del driver.
    - jp_in:            Entrada del joc de proves.
    - jp_out:           Sortida del joc de proves.
    - jp_explicat.txt:  Explicacio de les entrades de jp_in.txt i les seves sortides esperades.
    - README.md         Markdown amb l'operativa

Operativa:
  Execucio interactiva:

      java -jar DriverJugar.jar

    Es presenta un menu amb opcions per a utilitzar les diferents funcionalitats del programa.

  Execucio jocs de proves:

      java -jar DriverJugar.jar -q < jp_in.txt > sortida.txt

    Per a comprovar la validesa del resultat, executar:

      diff sortida.txt jp_out.txt

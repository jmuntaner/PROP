DriverEstadistiquesPartida

Arxius:
DriverEstadistiquesPartida.jar:  Executable amb del driver.
jp_in:             Entrada del joc de proves.
jp_out:            Sortida del joc de proves.
jp_explicat.txt:   Explicacio de les entrades de jp_in.txt i les seves sortides esperades.

Operativa:
  Execucio interactiva:

      java -jar DriverEstadistiquesPartida.jar

    Es presenta un menu amb opcions per a utilitzar les diferents parts de la classe.

  Excucio jocs de proves:

      java -jar DriverEstadistiquesPartida.jar -q < jp_in.txt > sortida.txt

    Per a comprobar la validesa del resultat, executar:

      diff sortida.txt jp_out.txt
